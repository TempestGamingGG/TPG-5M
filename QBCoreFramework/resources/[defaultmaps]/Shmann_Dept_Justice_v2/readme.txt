IMPORTANT:
If you don't own the RHPD, you have to enable the entityset "closed" to close the bridge.
You can activate the entityset serverside by following this links:

FiveM:
https://docs.fivem.net/natives/?_0x55E86AF2712B36A1
RageMP (_ENABLE_INTERIOR_PROP):
https://cdn.rage.mp/public/natives/#native_0x55E86AF2712B36A1
AltV: 
https://natives.altv.mp/#/0x55E86AF2712B36A1

You can also edit the Ymap "shmann_justice_mlo.ymap" with OpenIV to enable the Entityset.
Set the defaultentityset like this:

<defaultEntitySets>
     <Item>closed</Item>
</defaultEntitySets>

OR

simply use the "shmann_justice_mlo.ymap" in the mainfolder.
It's closed by default.
